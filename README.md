# 报价问答工作台 MVP

## 启动方式

在本目录执行：

```bash
python3 -m http.server 8766
```

然后访问：

```text
http://127.0.0.1:8766/
```

## 当前能力

- SKU 查询报价工作台：按公司级产品线、部署模式、关键词筛选 SKU。
- 报价草稿：选择 SKU 后录入数量、月数、人工询价金额，计算草稿小计和总额。
- 报价问答助手：输入客户场景后，基于当前筛选范围、语义场景、扩展关键词和 SKU 规则推荐可能需要的 SKU。
- 规则提示区：展示部署模式、周期、赠送、税号、询价、运维费、折扣、人天评审等已确认口径。
- 风险检查：自动提示人工询价、需求评审、审批、依赖项、赠送 SKU 等风险。

## 第一版边界

- 不自动判断最终折扣。
- 不自动决定商务最低价。
- 不自动确定运维费比例区间。
- 不自动估算实施/开发/集成人天。
- `目录价 = -` 的 SKU 只提示人工询价，并允许人工录入金额继续计算。
- 依赖项只提示“请另选择对应 SKU”，暂不自动加入报价明细。
- 税号阶梯已录入，但“按总税号落档还是按收费税号落档”仍保留为待确认口径。
